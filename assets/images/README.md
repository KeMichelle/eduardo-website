# Image Assets Directory

This directory contains placeholder and actual images for the EDUP Global Flex website.

## Structure

- `logo/` - Company logos and branding
- `products/` - Product images and thumbnails
- `team/` - Team member photos
- `banners/` - Hero section and promotional banners
- `icons/` - Custom icons and graphics

## Image Guidelines

- Logo: SVG format preferred, PNG fallback (min 300x300px)
- Products: High-quality images, 800x600px minimum
- Team photos: Professional headshots, 400x400px
- Banners: Hero images, 1920x1080px for optimal display
- Icons: SVG format for scalability

## Placeholder Information

All image components include fallback designs with:

- Gradient backgrounds using Ecuadorian colors
- Company name/logo text
- Appropriate icons and patterns
- Responsive scaling

## Usage

Images are imported in components and include error handling with graceful fallbacks to ensure the website remains functional even when images are missing.
